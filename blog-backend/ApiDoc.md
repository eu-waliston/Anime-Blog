## Post Router

#### GET
```
get all Posts: 

/blog/post

get a post by ID:

/blog/post/:id 

```

#### POST
```
Create a new Post:

/blog/post
```
#### PUT
```
Edit a Post:

/blog/post/:id
```

#### DELETE
```
Delete a post:

/blog/post/:id
```
